# Caractères spéciaux 

Au Clavier Windows : `Alt+Shift+U`, relâcher `Shift+U`, maintenir `Alt`, saisir le caractère **Décimal**, puis relâcher. 

Quelques caractères bien pratiques (avec le code décimal) :

- À : U+0192
- Ç : U+0199
- È : U+0200
- É : U+0201
- « : U+9134
- » : U+9135
